# What is Skillmer?
Skillmer is an app that is developed from a gamer for other gamers.
My goal is to create something similar to Instagram and Twitter but exclusive for gamers.
A platform where you can connect with other gamers, read and talk about games, consoles and all other stuff around gaming!

# In Developemnt
